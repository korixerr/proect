# Telegram Password Bot

Простой Telegram-бот на Python для генерации случайных паролей.

Бот принимает команду `/generate` и отправляет пользователю пароль заданной длины. Если длина не указана, используется значение по умолчанию - 12 символов.

## Возможности

- команда `/start`;
- команда `/generate`;
- генерация пароля длиной от 1 до 100 символов;
- использование строчных и заглавных букв;
- использование цифр;
- использование специальных символов `!@#$%^&*`.

## Требования

- Python 3.11 или выше;
- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather);
- библиотека `pyTelegramBotAPI`.

## Установка

1. Перейдите в папку проекта:

```powershell
cd C:\telegram-weather-bot
```

2. Создайте виртуальное окружение:

```powershell
python -m venv .venv
```

3. Активируйте виртуальное окружение:

```powershell
.\.venv\Scripts\Activate.ps1
```

4. Установите зависимость:

```powershell
python -m pip install pyTelegramBotAPI
```

## Настройка

В коде укажите токен Telegram-бота:

```python
TOKEN = "ваш_токен_бота"
```

Не публикуйте токен в GitHub или GitLab. Если токен попал в открытый доступ, перевыпустите его через BotFather командой `/revoke`.

## Запуск

```powershell
python bot.py
```

После запуска в консоли появится сообщение:

```text
Бот запущен. Ожидание сообщений...
```

## Использование

Откройте бота в Telegram и отправьте:

```text
/start
```

Для генерации пароля длиной по умолчанию:

```text
/generate
```

Для генерации пароля нужной длины:

```text
/generate 16
```

Пример ответа:

```text
Твой пароль:
aB7$kLm2Qx@9
```

## Ограничения

Длина пароля должна быть от 1 до 100 символов.

Если указать некорректное значение, бот отправит сообщение об ошибке.

## Как работает бот

1. Пользователь отправляет команду `/generate`.
2. Бот проверяет, указана ли длина пароля.
3. Если длина не указана, используется значение `12`.
4. Если длина указана, бот проверяет, что она находится в диапазоне от 1 до 100.
5. Функция `make_password()` случайно выбирает символы из заданного набора.
6. Готовый пароль отправляется пользователю в Telegram.
